ВЫЧИСЛЕНИЕ ОПРЕДЕЛЕННЫХ ИНТЕГРАЛОВ
----------------------------------

Данная программа предназначена для вычисления определенных интегралов и погрешностей.
Также она способна построить график этого интеграла.

Для получения данного ПО вам необходимо скачать архив и распаковать его.
Далее необходимо зайти в папку dist с файлом project.exe и запустить его.


ДОКУМЕНТАЦИЯ
------------

Документация по проекту находится в архиве с ПО


КОМПОНЕНТЫ
----------

Код ПО распределен по трем файлам:
project.py - содержит основной исходынй код
mydesign.py - содержит реализацию графического интерфейса
reshintegral.py - содержит функцию реализации метода Симпсона

Подключенные библиотеки и модули:
PyQt5 - библиотека для реализации графического интерфейса
https://www.qt.io/qt-for-python
numpy - предоставляет оптимизированные реализации вычислительных алгоритмов
https://numpy.org/
matplotlib - библиотека для реализации построения графиков
https://matplotlib.org/
sympy - библиотека для реализации математических вычислений
https://www.sympy.org/en/index.html
